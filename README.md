## 📌 Objective
To design and develop a dynamic To-Do List application using **HTML, CSS, and JavaScript**.  
The application allows users to add, edit, delete, and mark tasks as completed dynamically.

---

## 🛠️ Technologies Used
- HTML (Structure)
- CSS (Styling & Layout)
- JavaScript (DOM Manipulation & Logic)

---

## ✨ Features Implemented

- Add new tasks dynamically
- Prevent empty tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed (Strikethrough effect)
- Count pending and completed tasks
- Clean and user-friendly interface
- Permanent display of Name & Roll No on screen

---

## ⚙️ How to Run the Project

1. Download or open the project folder `dynamic_todo_list`.
2. Open `index.html` in any web browser.
3. Start adding tasks using the input field.
4. Use Edit/Delete buttons to manage tasks.
5. Mark tasks as completed using the checkbox.
